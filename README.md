# British-Airways-Tableau-Data-Analysis-Project
Tableau dashboard analyzing British Airways customer reviews. This project showcases data visualization skills by allowing users to dynamically explore review data based on various interactive metrics and filters.

#Project Goals

-Interactive Metric Selection: Users can easily switch between different review metrics (e.g., overall rating, food rating, entertainment rating) to gain diverse insights into customer feedback.
-Comprehensive Filtering Capabilities: The dashboard offers multiple filters including date range, country, seat type, traveler type, aircraft model, and continent, enabling detailed data exploration.
-Dynamic Visual Interactivity: All visuals on the dashboard act as filters, allowing users to click on specific countries, months, or aircraft to instantly update the displayed data.
